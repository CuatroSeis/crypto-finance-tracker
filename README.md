# crypto-finance-tracker
Real-time cryptocurrency dashboard with portfolio tracking and currency conversion

## Features
- 🔴 Live crypto prices (Bitcoin, Ethereum, and more)
- 📊 Historical price charts (7d / 30d / 90d)
- 💼 Personal portfolio tracker
- 💱 Currency converter (USD, EUR, ARS, BTC)
- 🌙 Dark / Light mode

## Tech Stack
- Vanilla JavaScript (ES6+ modules)
- HTML5 / CSS3
- [CoinGecko API](https://www.coingecko.com/en/api) — free, no key needed
- [ExchangeRate API](https://www.exchangerate-api.com/) — free tier
